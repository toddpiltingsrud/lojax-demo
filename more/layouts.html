A Layout Strategy

Here's an easy trick.

Instead of placing jx-panel atttributes on every piece of content, create a layout that checks the request for an "X-Requested-With" header with a value of "XMLHttpRequest". Most self-respecting AJAX libraries use this convention (including jQuery).

If this header exists, just render the relevant jx-panel. If not, render the whole page. This way any page can be re-used to render in full or in part.
